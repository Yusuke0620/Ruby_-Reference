- ## 要素を追加(配列の最後)
「<<」要素を使い追加
<a name="add_end_of_array"></a>
```ruby
array = [0]
array << 1

p array #[0, 1]
```

- ## 要素を削除(配列の先頭から)
`delete_at` メソッドを使い削除
<a name="delete_from_start_of_array"></a>
```ruby
array = [1, 2, 3]
array.delete_at(0)
p array # [2, 3]
